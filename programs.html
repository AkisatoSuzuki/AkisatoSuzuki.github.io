bayesdtm: A Bayesian Decision-Theoretic Model to Compute Expected Losses with and without an Intervention

Near completion

This package implements a Bayesian decision-theoretic model to compute expected losses given an intervention (in causal inference senses) and those without it, and returns the plot of these expected losses (currently available only for outputs from a Bayesian logistic regression).

ccdfpost: Plot a Complementary Cumulative Distribution Function for the Posterior Samples of a Causal Effect

Near completion

This function produces the plot of a complementary cumulative distribution function for the posterior samples of a causal effect.
